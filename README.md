# Phishing Analysis Project

Analyze phishing patterns with Python. Collects data, detects suspicious URLs, calculates risk scores, and generates security training materials.

## Quick Start
1. Install: `pip install -r requirements.txt`
2. Run: `jupyter notebook phishing_analysis.ipynb`

## Features
- Fetch phishing data from APIs
- Analyze URL patterns and keywords
- Calculate risk scores for threats
- Generate training materials
- Create visualizations and reports

## Outputs
- Risk assessment reports
- Phishing awareness guides
- Executive summaries
- Interactive dashboards

## Requirements
Python 3.8+ with pandas, matplotlib, seaborn, requests
